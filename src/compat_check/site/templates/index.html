How to read this

Each cell shows what percentage of SD-6 feature tests compile successfully at that C++ standard. Click any cell to see the per-feature breakdown. "Best" is the highest percentage across all standards for that platform.

Status classifications

supported — SD-6 macro defined and feature compiles
unreported — feature compiles but macro not defined
macro_lies — macro claims support but compile fails
unsupported — feature does not compile

Both supported and unreported count as working features. macro_lies means the toolchain advertises a feature it can't deliver — a common issue with embedded standard libraries.
